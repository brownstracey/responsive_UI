About

This app is a small planner for students. It’s made with just HTML, CSS and JavaScript (no frameworks!).

It’s supposed to show off things like responsive design, validation and regex (whatever that is lol).

Contact me on GitHub or send me an email at you@example.com.

Regex Examples:

  • Title: ^\S(?:.*\S)?$
  • Duration: ^(0|[1-9]\d*)(\.\d{1,2})?$
  • Date: ^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$
  • Tag: ^[A-Za-z]+(?:[ -][A-Za-z]+)*$
  • Duplicate words: \b(\w+)\s+\1\b
